AISearchViewerLite is a lightweight, read-only Excel viewer designed for
safe inspection and instant web search from cell contents.

This tool never modifies Excel files.
All cells are treated strictly as strings to avoid unintended data conversion.

Key features

Read-only viewing of .xlsx / .xlsm files (non-destructive)

All cell values displayed as strings (no date/number auto-conversion)

Right-click on a cell to search its content via web browsers

Search string normalization before querying

Column header click sorting (ascending / descending)

Automatic column width adjustment (per column)

Zebra row styling for readability

Cell-level highlight with border only (no row-wide highlight)

External configuration via INI files

Keyboard shortcuts for efficient operation

This is the first public release of AISearchViewerLite.


AISearchViewerLite は、Excel（.xlsx / .xlsm）ファイルを
安全に閲覧し、セル内容を即座に検索するための軽量ビューアです。

Excel ファイルは一切変更されず、
すべてのセルは文字列として扱われるため、
日付や数値の意図しない変換が起きません。

主な機能

Excel ファイルを読み取り専用で表示（非破壊）

すべてのセルを文字列として表示

セル右クリックから即座に Web 検索

検索前の文字列正規化処理

列ヘッダクリックによる昇順／降順ソート

列幅の自動調整（列ごと）

行のゼブラ表示

行全体ではなく「セル単位」の枠ハイライト

INI ファイルによる外部設定

キーボードショートカット対応

本リリースは AISearchViewerLite の初公開バージョンです。
